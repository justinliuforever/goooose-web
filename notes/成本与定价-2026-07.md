# Goooose 成本与定价基线（2026-07）

**方法**：单价来自各服务官方页 2026-07 逐家核实；用量来自生产库 `usage_events` 真实成本遥测（30 个 run、363 条计费事件、2026-07-05 至 07-10，price_version `2026-07-promo`）+ `proxy_sessions` 真实字节 + `pipeline_runs` 真实时长。标「实测」的是生产遥测直接读出；标「模型」的是用单价 × 典型用量推算（对应场景生产样本不足）。

**一句话**：固定月费约 **$79**，在 50 人闭测规模下所有服务都落在固定/免费档不产生超额；单件内容边际成本很低（一条小红书笔记分析 ≈ **$0.026 实测**、一条 10 分钟 YouTube 视频分析 ≈ **$0.15–0.23 模型**、一篇短稿 ≈ **$0.018 实测**）；一个跑满的 300 分钟/月配额池，我们的变动成本约 **$1–7**，取决于用户有多依赖 YouTube。真正的成本地板是固定月费，不是按量。

三条要注意：
1. **Deepgram 低估 79% 目前无实际影响** —— 生产至今全部 ASR（348 分钟）都走了 Qwen（中文内容），Deepgram 一次没用上。等 YouTube 英文内容上量才会真实发生，届时修正单价。

3. **代理是唯一明显偏贵的单价**，也是最容易砍的成本杠杆（换供应商即可 2–4×）。

## 三、每类内容单件成本（核心）

> 变量成本 = LLM + Vision + ASR + 抓取 + 代理（Trigger compute 另计，见第四节）。

### Clerk 分析

| 单件 | 成本 | 依据 |
|---|---|---|
| 一条**小红书笔记**（混合）| **$0.026** 实测 | 计费 run 实测：20 笔记 $0.81 / 20 笔记 $0.53 / 10 笔记 $0.18 → $0.018–0.041/条 |
| ├ 图文笔记 | ~$0.02 | 多图 Vision + Pro（标题正文短）+ TikHub，无 ASR |
| └ 视频笔记 | ~$0.03 | Qwen ASR（均 3.8min ≈ $0.008）+ 封面 Vision + Pro + TikHub |
| 一条**10 分钟 YouTube 视频**（模型）| **$0.15–0.23** | 生产 YT 视频均 11 分钟；成本大头是代理+ASR |
| ├ 有字幕（约六成）| ~$0.15 | 代理 ~$0.13 + Pro $0.011 + Vision $0.004，无 ASR |
| └ 需转写 | ~$0.23 | 再加 Deepgram 11min × $0.0077 ≈ $0.085 |

小红书条目成本只有 YouTube 的 1/6 左右，因为**不走代理、也几乎不走 Deepgram**。YouTube 的成本被代理主导（约占单条一半），换便宜代理能把 YT 单条压到 ~$0.08–0.12。

| 操作 | 成本/次 | 样本 | 说明 |
|---|---|---|---|
| Muse 巡视（出选题）| **$0.059** | n=3 | 每次约出 13 条选题，含 ASR + Pro；按对标账号数放大 |
| Poet 短稿 | **$0.018** | n=8 | Pro 写稿 + 中文口语化；in 31k / out 10k token |
| Poet 圣经生成 | **$0.003** | n=1 | 纯 Pro，最便宜 |
| Poet 圣经文件导入 | **$0.097** | n=4 | Vision 重（文档逐页转写），单次最贵的生成类 |

| Clerk 单视频分析 | **$0.008** | n=2 | Pro，无 ASR/代理 |

### 单次成本结构（各资源占比，Clerk 频道分析实测）

Vision **$0.217/run** + ASR **$0.320/run** + LLM **$0.068/run** + 抓取 $0.004/run。即 **ASR+Vision 占 ~80%，DeepSeek LLM 只占 ~17%**。全局累计成本第一名是 **Claude Vision（$1.02）**，其次 Qwen ASR（$0.73）、DeepSeek Pro（$0.41）、Flash（$0.18）。这与「AI 写稿成本高」的直觉相反——真正贵的是看图和听音，写字很便宜。

## 五、每配额分钟的真实成本（定价锚点）

产品按分钟计费（300 分钟/月单池）。把「卖出去的一分钟」换算成我们的 COGS：

| 操作 | 扣配额分钟 | 真实 COGS | 每配额分钟 COGS |
|---|---|---|---|
| 圣经生成 | 5（固定）| $0.003 | **$0.0006** |
| 自定义选题 | 3（固定）| $0.008 | $0.0027 |
| 短稿（目标 5min）| 5 | $0.018 | $0.0036 |
| 单视频 | 2（固定）| $0.008 | $0.004 |
| 小红书图文 | 5（固定）| $0.02 | $0.004 |
| 小红书视频 | ≈4（实际时长）| $0.03 | $0.0075 |
| 圣经文件导入 | 10（固定）| $0.097 | $0.010 |
| YouTube 10min 有字幕 | 10（实际时长）| $0.15 | $0.015 |
| YouTube 10min 需转写 | 10（实际时长）| $0.23 | **$0.023** |

**每配额分钟 COGS 跨度 $0.0006 ~ $0.023，差 ~38 倍。** 贵的只有 YouTube 视频分析这一种「分钟」；写稿/选题/圣经/小红书的「分钟」都很便宜。

一个跑满的 300 分钟池对我们的变动成本：
- 全用于 YouTube 需转写视频（最坏）：300 × $0.023 ≈ **$6.9/月**
- 全用于小红书/写稿/圣经（最省）：300 × ~$0.004 ≈ **$1.2/月**
- 现实混合：约 **$2–4/月变动成本**

## 七、定价含义与建议

1. **固定成本是地板，不是按量。** $79/月在 ¥99（~$14）定价下 **约 6 个付费用户即回本**，¥199 下约 3 个。闭测阶段成本几乎全是这 $79，按量可忽略。

2. **毛利空间充足。** 即便重度 YouTube 用户（COGS ~$8），¥99/月定价毛利 ~45%、¥199/月 ~70%；小红书为主的用户毛利 90%+。

3. **三个变动成本杠杆**（按性价比排序）：
   - **换代理供应商**：$6/GB → $1.5–4/GB，直接砍 YouTube 单条成本约一半（YouTube 单条一半是代理）。最省力见效最大。
   - **Vision 限量**：Claude Vision 是全局第一大成本。小红书多图笔记（最多 9 张）叠图分析吃 token；可评估只看封面或只对 top-N 笔记做视觉。
   - **DeepSeek 排非峰时**：规避疑似 2× 峰时加价（影响面小，LLM 仅占 ~17%）。

4. **配额定价的结构性提醒**：一个「分钟」的成本因内容类型差 38 倍。300 分钟单池对小红书/写稿用户是「送」，对重度 YouTube-转写用户才逼近成本。若未来分层定价，可考虑：YouTube 视频分钟按更高倍率折算，或对 YouTube 分析单独限量/加价——避免少数重度 YT 用户拉低整体毛利。

5. **规模化时的成本迁移**（beta 后再看）：用户上量后 Supabase compute、Vercel 传输、Trigger compute 会陆续开始产生超额；Trigger 月费过 $200 时按既定策略迁自托管。这些都在闭测之后。
