# v0.8 — 搬砖小鹅 Goooose

2026-07-18 · Beta

## 新功能

### 抖音

- 添加自有账号 / 导入对标：主页链接、`v.douyin.com` 分享短链、整段分享口令文本都能直接粘
- Clerk 拆解：最新作品 / 热门作品（按互动分）/ 指定链接三种模式，支持时间范围筛选（此前 YouTube 独占）
- 视频转写双路径：原声音轨直取（命中时免下载视频），否则最低码率下载抽音轨；封面视觉分析同小红书
- 图文（图集）作品完整支持：整组图进视觉分析，SOP 按「帖子N（图文）」引用
- 爆款 SOP 附评论区共鸣分析（top 100 条评论）——YouTube 之外第二个有评论洞察的平台
- Muse 竞品巡视 + 选题生成；内容类型过滤（全部 / 仅视频 / 仅图文）现同时作用于小红书与抖音对标
- Poet 自定义选题支持抖音链接做参考素材；选题 → 写稿全旅程可用
- 单条深拆（视频作品）

### statement

- 抖音不公开播放数：列表与排序用加权互动分（点赞 + 收藏×2 + 评论×3 + 分享×5），UI 明确标注「互动分」，分析提示词同步声明口径，不冒充播放量
- 抖音无独立标题：文案前 80 字作展示标题，全文进入分析
- 视频播放链接约 60-90 分钟过期：管道内即时获取，不落库；封面图有效期长可直接缓存

### 平台与后台

- 新版本首次打开弹「更新说明」（按用户存版本比对，只弹一次，跨设备一致）
- 管理后台重构：单页拆成「准入 / 用户 / 用量·成本 / 运维」四个分段，每段只查自己的数据
- 运维监控（全新）：跨用户看进行中 / 卡住 / 24h 失败的任务，卡住的任务高亮
- 后台补：用户「版本」列（谁没看过新版）、用户 / 申请表搜索筛选、申请表总数 + 分页
- 兑换码前缀 `SING-` → `GOOSE-`（旧码仍可兑换）

## 质量和之前的修复

- TikHub 瞬时故障重试预算加宽（实测抖动可自愈）
- 修复的旧问题：实时日志按钮嵌套的 hydration 报错（v0.7 起存在）；Muse 选题来源一律被错标 YouTube；图文孤儿选题恢复被视频字数门槛卡死
- Clerk 抖音 / 小红书两条管线的「分析→视觉→落库」核心合并为单一实现（消除复制、防未来漂移）
